# SS-HOuses Frontend

This project was build using React 17.0.1

## Installation

Clone this repo, open the `SS-Houses` folder and type the next commands in your terminal.

# Usage

- `npm install`
- `npm start`
- Go to http://localhost:9000/

# Test

- `npm test`

# Test Coverage

- `ng test --no-watch --code-coverage`
